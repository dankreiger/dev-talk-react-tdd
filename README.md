# Dev Talk TDD + React

#### Prerequisites

- apiKey from [https://newsapi.org/](https://newsapi.org/)
- enter it in `presentation/index.js`

#### Usage



- Install dependencies

  ```sh
  npm install
  ```

- Start Server

  ```sh
  npm start
  ```
